# HYBRID Engineering Standards

Fonte normativa corporativa da HYBRID Tecnologia Inteligente para arquitetura, engenharia, segurança, dados, APIs, IA, UX, DevSecOps e documentação.

## Escopo

Este repositório estabelece padrões reutilizáveis por todos os produtos da HYBRID sem incorporar regras de negócio específicas.

## Fronteiras

- `hybrid-standards`: normas corporativas.
- `micael-project`: governança e coordenação do Programa MICAEL.
- `micael-book`: documentação viva da família MICAEL.
- repositórios de produto: domínio, implementação, testes e operação próprios.

## Estrutura inicial

- `hea/`: HYBRID Enterprise Architecture.
- `hes/`: HYBRID Engineering Standard.
- `adr/`: decisões arquiteturais corporativas.
- `rfc/`: propostas em discussão.
- `architecture/`, `api/`, `database/`, `security/`, `ai/`, `devops/`, `ux/`: padrões especializados.
- `templates/`: modelos oficiais.

## Governança

Mudanças normativas seguem o fluxo RFC -> revisão -> ADR -> publicação -> adoção pelos projetos.
