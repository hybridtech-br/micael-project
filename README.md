# MICAEL

Plataforma inteligente para gestão condominial, segurança eletrônica, monitoramento por câmeras, reconhecimento facial, leitura de placas e controle de acesso.

## Produtos

- **MICAEL Condo** — administração de condomínios, moradores, visitantes, veículos, comunicação, ocorrências, reservas e relatórios.
- **MICAEL Security** — gerenciamento de câmeras, análise inteligente de vídeo, reconhecimento facial, leitura de placas e controle de portas, portões, cancelas e catracas.
- **MICAEL Core** — identidade, permissões, multi-tenant, auditoria, notificações, arquivos, integrações e serviços compartilhados.
